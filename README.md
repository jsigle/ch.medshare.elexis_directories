Adress-Suche_js / Aktualisierung von ch.medshare.elexis_directories
===================================================================

Version des "medshare directories" Plugins für Elexis, jeweils in 2010 und 2012 von mir aktualisiert und erweitert.

Nach Änderungen im Ausgabeformat der Website, die das Plugin befragt, hatte dieses jeweils nicht mehr funktioniert.
Die Erweiterungen der Funktionalität betreffen das Format der übernommenen Daten; die jüngste Fassung entschlüsselt
auch den codiert gelieferten Eintrag für die e-Mail-Adresse und überträgt diese ins entsprechende Feld in Elexis.

Ausführliche Informationen auf: http://www.jsigle.com/prog/elexis#Adress-Suche_js
